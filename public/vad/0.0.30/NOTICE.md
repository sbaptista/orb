# Self-hosted Silero VAD browser assets

These files are pinned for Orb's shadow-only voice activity classifier:

- `silero_vad_v5.onnx` — Silero VAD model, MIT License, copyright Silero Team.
- `vad.worklet.bundle.min.js` — `@ricky0123/vad-web` 0.0.30, ISC License, copyright ricky0123.
- `ort-wasm-simd-threaded.wasm` and `ort-wasm-simd-threaded.mjs` — ONNX Runtime Web 1.27.0, MIT License, copyright Microsoft Corporation.

Upstream license texts remain available in the installed npm packages. These
assets are observational only: they do not gate, suppress, replace, or transmit
microphone audio.
